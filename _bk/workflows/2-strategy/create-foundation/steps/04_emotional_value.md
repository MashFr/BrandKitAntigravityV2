  <action>
    Dernière étape clé : La **Proposition de Valeur Émotionnelle**.
    
    Demande : "Quelle est la transformation psychologique du client ?"
    - **Avant [Marque]** : Comment se sent-il ? (Frustré, perdu, invisible...)
    - **Avec [Marque]** : Comment se sent-il ? (Confiant, puissant, apaisé...)
    
    ---
    
    **Synthèse Finale** :
    
    Compile TOUTES les informations collectées dans les étapes précédentes pour remplir le template `@brand-strategy-template.md`.
    
    Remplace les placeholders par les réponses validées :
    - {{PROJECT_NAME}}
    - {{WHY}}, {{HOW_1/2/3}}, {{WHAT}}
    - {{ARCHETYPE_PRIMARY}}, {{ARCHETYPE_PRIMARY_DESC}}
    - {{ARCHETYPE_SECONDARY}}, {{ARCHETYPE_SECONDARY_DESC}}
    - {{FEAR}}, {{DESIRE}}
    - {{VALUE_1/2/3}}, {{VALUE_1/2/3_PRINCIPLE}}, {{VALUE_1/2/3_IMPACT}}
    - {{BEFORE_STATE}}, {{AFTER_STATE}}
    
    Génère le fichier `brandkit_output/01_foundation_strategy.md`.
    Présente le résultat à l'utilisateur.
  </action>
</step>

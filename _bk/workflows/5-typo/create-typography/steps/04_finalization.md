<step id="04_finalization">
    <action>
        Finalisons le guide typographique.
        
        1.  Établis les règles d'interlignage (Line-height) et les Do's / Don'ts typographiques.
        2.  Remplis le template `@typography-template.md`.
        
        Remplace les placeholders :
        - {{PROJECT_NAME}}, {{TYPE_STYLE}}, {{TYPE_RATIONALE}}
        - {{TITLE_FONT_...}}, {{BODY_FONT_...}}, {{ACCENT_FONT_...}}
        - Tableau Hiérarchie : {{H1_...}}, {{P_...}}, etc.
        - {{LINE_HEIGHT_H}}, {{LINE_HEIGHT_P}}
        - {{DO_1..2}}, {{DONT_1..2}}
        
        Génère le fichier `brandkit_output/04_typography.md`.
        Conclus en rappelant que la typographie est la voix du texte.
    </action>
</step>

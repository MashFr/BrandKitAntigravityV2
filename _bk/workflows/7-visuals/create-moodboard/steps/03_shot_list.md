description: Step 3 - Création de la Shot List (Hero, Portrait, Detail)
---

<step id="03_shot_list">
    <action>
        Génère les 3 prompts maîtres pour le Moodboard.
        
        1.  **Hero Shot (Paysage --ar 16:9)** :
            - Une image qui capture toute l'essence de la marque.
            - Souvent un plan large (Wide shot).
            
        2.  **Portrait ou Produit (Portrait --ar 4:5)** :
            - Focus sur l'humain (le client cible) OU le produit.
            - Éclairage studio ou contextuel fort.
            
        3.  **Texture / Détail (Carré --ar 1:1)** :
            - Macro shot. focus sur la matière, le grain, la couleur.
            - Peut servir de fond abstrait.
            
        Écris le prompt complet pour chaque shot.
    </action>
</step>

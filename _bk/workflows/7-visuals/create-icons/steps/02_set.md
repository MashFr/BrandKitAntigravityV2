---
description: Step 2 - Sélection du Set d'Icônes Maîtres
---

<step id="02_set">
    <action>
        Basé sur le **Brand Brief** (00), identifie les 5 concepts clés qui nécessitent une icône propriétaire.
        
        Par exemple, pour une application de santé :
        - Cœur (Vitalité)
        - Bouclier (Sécurité)
        - Éclair (Énergie)
        - ...
        
        L'objectif est d'éviter les icônes "stock" et de créer un set reconnaissable.
    </action>
</step>

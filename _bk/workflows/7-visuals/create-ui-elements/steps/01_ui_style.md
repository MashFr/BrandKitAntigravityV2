description: Step 1 - Définition du Style UI (Aesthetic Direction)
---

<step id="01_ui_style">
    <action>
        Analyse la **Typographie** (04), les **Couleurs** (03) et l'**Univers Graphique** (05) pour définir l'apparence de l'interface.
        
        Défins :
        1. **Le Courant Esthétique** : Minimalisme pur, Glassmorphisme (transparences), ou Neumorphisme (ombres douces) ?
        2. **Le Traitement des Bords** : Très arrondis (apps modernes/lifestyle) ou angles vifs (tech/expert) ?
        3. **La Profondeur** : Est-ce une interface plate (2D) ou jouant sur les couches et les ombres (3D) ?
    </action>
</step>

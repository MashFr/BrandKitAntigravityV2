<step id="02_components">
    <action>
        Basé sur le **Brand Brief** (00), identifie les composants d'interface les plus critiques pour la marque.
        
        Par exemple :
        - Pour un E-commerce : Fiches produits et boutons d'achat.
        - Pour une SaaS : Dashboards, cartes de statistiques, menus de navigation.
        - Pour un média : Cartes d'articles et champs de recherche.
    </action>
</step>

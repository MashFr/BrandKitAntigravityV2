<step id="03_prompting">
    <action>
        Génère les prompts pour créer ces concepts d'interface.
        
        **Technique "UI Prompting"** :
        - Utilise des tokens comme `user interface design`, `modern web application ui`, `clean visual hierarchy`, `consistent design system`.
        - Précise le style visuel : `glassmorphism`, `pastel gradients`, `dark mode` ou `clean light mode`.
        
        Structure :
        `A high fidelity UI design for [COMPONENT] of a [TYPE OF APP], [STYLE], [COLORS], [LAYOUT], dpreview style, extremely clean --ar 16:9`
    </action>
</step>

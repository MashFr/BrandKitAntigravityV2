---
description: Step 3 - Contrôle d'Accessibilité
---

<step id="03_accessibility">
    <action>
        Sur la palette l'option choisie, nous allons définir les neutres et vérifier l'accessibilité.
        
        1.  **Définis les Neutres** : Un fond clair (pas blanc pur) et des couleurs de texte.
        2.  **Test de Contraste** :
            - Estime les ratios de contraste (approximatifs) pour :
            - Texte Principal sur Fond Clair.
            - Texte Blanc sur Couleur Primaire.
            - Texte Blanc sur Couleur Secondaire.
        
        SI le contraste est insuffisant (< 4.5:1), AJUSTE la luminosité des couleurs pour les rendre conformes WCAG AA.
        Explique les ajustements faits si nécessaire.
    </action>
</step>
